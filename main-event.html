主場活動

兩日集結兒童劇團、親子表演、氣球魔術、 人氣藝人與樂團演出，現場還有每日摸彩活動， 邀請大家一起來三鶯度過熱鬧又好玩的夏日時光。

ABOUT THE EVENT

兩天不同內容，
從親子歡樂到音樂舞台

8月1日以親子表演為主軸，安排蘋果兒童劇團、 腹語魔術、樂團、氣球表演，以及波力救援小英雄、 發桐醬與東森 YOYO 見面演出，適合親子家庭一起參加。

8月2日由開幕式揭開序幕，接續泡泡秀、小丑氣球秀、 掌中劇、音樂演出與人氣藝人舞台，從白天一路熱鬧到傍晚。 每日最後皆安排摸彩活動，現場將抽出多項實用好禮。
